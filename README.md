🔐 Authentication Security Labs

This repository contains my practice labs and notes related to Authentication vulnerabilities in web applications. 
The goal of this repository is to document different authentication flaws, understand how attackers exploit them, 
and learn how developers can prevent them.

These labs are part of my journey in web security, penetration testing, and bug bounty hunting.
📚 Topics Covered

This repository focuses on common authentication-related vulnerabilities, including:

Username Enumeration
Password Brute Force Attacks
Broken Authentication Logic
Authentication Bypass
Login Rate Limiting Issues
Multi-Factor Authentication (MFA) Bypass
Session Handling Weaknesses
Credential Stuffing
Predictable Usernames or Passwords

Each lab explains how the vulnerability works, how it can be exploited, and how it can be mitigated.


🧰 Tools Used

During these labs, the following tools are commonly used:
Burp Suite – Intercepting and modifying HTTP requests
Burp Intruder – Performing automated brute-force attacks
Browser Developer Tools – Inspecting requests and responses
Wordlists – For username and password testing
